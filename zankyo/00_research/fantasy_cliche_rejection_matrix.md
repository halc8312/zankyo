# ファンタジー・クリシェ拒否マトリクス

## 作品別観察メモ（25作）
1. **ロード・オブ・ザ・リング**: 種族本質化と使命中心。土地の傷は広いが、力の担い手は選抜される。  
2. **氷と炎の歌**: 権力の腐敗は厚いが、なお貴族家の視点比率が高い。  
3. **ミストボーン**: 能力設計は明晰だが、制度より操作体系へ注意が寄る。  
4. **風の名前**: 才能集約型主人公が世界の密度を吸い寄せる。  
5. **時の車輪**: 予言と選定が巨大化し、共同体の細部が後景化しやすい。  
6. **ゲド戦記**: 言葉と均衡の思想は強いが、個人修養が中心。  
7. **黒い会社**: 兵士の目線は有効だが、批評より消耗の感触が前面に出る。  
8. **ファースト・ロウ**: 冷笑が制度分析の代替になりがち。  
9. **マラザン**: 規模は大きいが、読者の参入障壁も大きい。  
10. **ティガナ**: 地名記憶と政治が近く、本作の比較対象として重要。  
11. **ゴーメンガースト**: 儀礼に縛られた屋敷社会の息苦しさが有効。  
12. **Prince of Nothing**: 深層構造は濃いが、虚無へ傾きやすい。  
13. **The Acts of Caine**: 階級意識と見世物化された暴力の批評が参考になる。  
14. **The Traitor Baru Cormorant**: 植民地支配と会計制度の接続が鋭い。  
15. **The Poppy War**: 戦争犯罪を直接的に扱うが、惨禍が能力劇へ回収されがち。  
16. **The Actual Star**: 時間をまたぐ残りかすの扱いが参考になる。  
17. **Piranesi**: 空間記憶と孤独な記録実務が有効。  
18. **Mexican Gothic**: 土地記憶と身体汚染の接続が強い。  
19. **The Fifth Season**: 抑圧制度と能力管理の接続が参考になる。  
20. **Children of Blood and Bone**: 表層借用の危険性を考える警告例。  
21. **Spinning Silver**: 労働と取引を能力条件へ結ぶ点が有効。  
22. **The House in the Cerulean Sea**: 慰撫に寄りすぎると歴史の傷が抜ける。  
23. **鹿の王**: 疫病、政治、生態を結ぶ手つきが重要。  
24. **精霊の守り人**: 政治正統性と土地の関係づけが参考になる。  
25. **十二国記**: 統治理念の検証が濃いが、適任者論の吸引力も強い。

## クリシェ拒否マトリクス
| クリシェ | よくある形 | 問題点 | 『残響』での拒否方針 |
|---|---|---|---|
| 選ばれた一人 | 予言、血筋、天啓 | 歴史の負荷が個人へ集中する | 能力は土地残響で発生し、個人は管理者か被拘束者にすぎない |
| 血筋が力を決める | 王家の末裔、隠された家系 | 支配を自然化する | 居住歴、葬送参加、労働履歴が発現条件 |
| 種族本質化 | エルフはこう、ドワーフはこう | 人間集団を固定化する | 地方差・階層差・職能差を優先し、種族分類を置かない |
| 王政復古が救済 | 正しい王の帰還 | 制度批判を避ける | 土地管理制度と記憶政策を変えない限り問題は残る |
| 冒険者ギルド万能 | 依頼掲示板で世界が回る | 労働と行政が薄い | 渡りは無秩序な個人ではなく、危険な仲介職能集団 |
| 能力は個人鍛錬で完成 | 修行で上達 | 土地の倫理が消える | 土地から離れると減衰し、共同体の手順に縛られる |
| 能力はわかりやすい数式 | 属性とランクで整理 | 社会制度が添え物になる | 強度モデルは持つが、法・葬制・記憶管理と連動させる |
| 悲劇は主人公の燃料 | 家族の死で覚醒 | 被害の消費になる | 悲劇は地域制度を変え、後始末労働を生む |
| 暗いだけで批評した気になる | 誰も報われない | 暗さが分析の代わりになる | 加害構造、利得主体、補修不能な傷を具体化する |
| 戦争が世界の中心 | 歴史の大半が戦役 | 生活史が薄くなる | 葬り、疫病、徴税、開墾、移住も歴史の中心に置く |
| 貴族だけが歴史を動かす | 王侯将相の視点固定 | 庶民の制度負担が消える | 墓守、渡り、書記、看病人、土木労働者を主要視点化 |
| 辺境は空白 | 地図の端の未開地 | 中心の都合でしか見えない | 辺境は複数権力の課税と記憶が重なる接触地帯 |
| 宗教は雰囲気装置 | 祭礼の飾り | 制度と倫理が抜ける | 埋葬、婚姻、土地禁忌、証言儀礼に直結させる |
| 文化は衣装で表現 | 服、武器、料理だけ違う | 表層借用になる | 匂い、労働、法、葬送手順、語彙の差で示す |
| 都市は一枚絵 | 巨大で雑然 | 生活導線がない | 水、死体処理、市場臭、夜警、墓地不足まで書く |
| 田舎は素朴で善良 | 支え合う共同体 | 監視と暴力を消す | 相互扶助と排除を同時に書く |
| 旅は自由 | 道は開かれている | 通行権と検問が消える | 渡りは宿権、井戸権、墓地規則に縛られる |
| 記憶は一つ | 国民全員が同じ過去を語る | 争点が消える | 公式史、民間史、敗者史、異端史を分岐させる |
| 和解で終わる | 謝罪の一言で収束 | 物的補償が消える | 遺骨返還、土地返還、名簿訂正がない限り終わらない |
| 悪の帝国 | 単純な加害国家 | 倫理の厚みが薄い | 利得主体を階層別に分け、被害と加害の重なりを書く |
| 古代文明の遺産万能 | 失われた超技術 | 現在社会の努力が不要になる | 残響は過去の傷だが、現代の維持労働がないと機能しない |
| 農村は背景 | 都市と城だけ詳細 | 食と土地の重みが薄い | 灌漑、地代、皮膚病、収穫暦を中心要素にする |
| 地図境界が固定 | 国境線で秩序完結 | 実際の接触が見えない | 市場、祭礼、難民路、埋葬圏で重層化する |
| 用語が既存作品寄り | mage, mana など | 独自性が消える | 残響、渡り、逆残響など歴史起源の語彙へ置換 |
| すべて説明し尽くす | 設定辞典的過剰明快さ | 史料差と沈黙が消える | 異説、欠損、誤記を意図的に残す |

## 実装メモ
- 主人公級人物でも、土地残響の外では弱まる前提を崩さない。  
- 事件を書くときは、誰が片付け、誰が記録し、誰が忘却で得をするかを同時に書く。  
- 各地方ファイルに少なくとも一つ、葬送・徴税・移動制限・証言の要素を入れ、上表の拒否方針に接続する。
