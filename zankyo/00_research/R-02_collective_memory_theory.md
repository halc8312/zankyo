# R-02 集合記憶理論

**ID**: R-02

**反映先ファイル**: `01_world_core/world_ethics.md`, `02_history/chronicle_official.md`, `06_conflicts/truth_regimes.md`

## テーマ要約
Halbwachs が示した集合記憶は、個人の頭の中にある思い出ではなく、家族、宗教団体、学校、軍、村落名簿のような枠組みによって保たれる再構成である。Nora と Connerton を併せて読むと、記憶は場所と儀礼に保存されるだけでなく、保存される形式の選別そのものが権力作用になる。

## 主要論点
- 記憶は事実の総量ではなく、誰が繰り返し語る権利を持つかで形を変える。
- 墓地、祝日、慰霊碑、校歌、戸籍台帳は記憶媒体である。
- 公式史は沈黙の管理でもあり、語られない死者が必ず残る。
- 身体技法としての追悼、行列、黙祷、供物配置が記憶を持続させる。
- 複数共同体が同じ事件を別々に記憶すると、境界線そのものが政治化する。

## 転用可能な構造
残響世界では、同一事件から複数の残響系統が生じる理由を「記憶枠組みの差」で説明できる。国家は戦没者台帳と慰霊式で統一された残響を育てようとし、遺族会は私的な呼び名と台所の供え物で別系統を残し、敗者は無名墓と口承で断片的だが執拗な残響を保持する、という三層構造が有効である。

## 浅借りへの注意
「人々は忘れない」と書くだけでは浅い。誰が忘却を命じ、誰が覚え続ける費用を払い、どの場所が清掃され、どの遺骨が数えられないのかまで制度として落とす必要がある。
