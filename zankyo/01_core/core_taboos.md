# core_taboos

## 概要
禁忌は道徳より先に労務安全と残響管理のために作られた。破れば祟るからではなく、土と身体の癖を乱し、争いと損失を招くから禁じられる。

## 詳細
| ID | 禁忌 | 実際の理由 |
|---|---|---|
| TAB-01 | 故郷の外へ埋めるな | 地所帰属と残響継承が曖昧になる |
| TAB-02 | 飽和地へ証札なく入るな | 集団症候と証拠汚染を防ぐ |
| TAB-03 | 惨地を加害者名で呼ぶな | 責任が地名へ吸われるのを避ける |
| TAB-04 | 生地の土を持ち出すな | 居住権と地歴の混同を防ぐ |
| TAB-05 | 飽和畑を弔いなく耕すな | 作業事故と感受暴走を防ぐ |
| TAB-06 | 死日同日に髪を切るな | 身体の切断感覚が残響に同期する |
| TAB-07 | 水門役人は夜更けに鍵を数えるな | 鍵の音が記憶誘発となる |
| TAB-08 | 渡りを井戸番にするな | 多地点干渉で誤判が増える |
| TAB-09 | 骨端日に炉骨残響を起こす炉を炊くな | 飽和上書きを招く |
| TAB-10 | 旱路で空の水皮を振るな | 渇歩残響を強める |
| TAB-11 | 舟葬歌を陸で完唱するな | 海哭残響が陸に移ると信じられる |
| TAB-12 | 証言を書いた手をその日のうちに洗うな | 文字の責任を身体へ留める |
| TAB-13 | 他郷の葦で棺を編むな | 地域識別が混じる |
| TAB-14 | 飢饉年の塩を婚資に使うな | 欠乏の記憶が家に入る |
| TAB-15 | 飽和地の土で子の足型を取るな | 幼児感受が強すぎる |

## 生活への影響
禁忌は警吏の取締よりも、嫁入り、共同作業、葬列、出市の場面で重く働く。違反者はすぐ罰されなくとも、共同井戸や皮干し棚から外される。

## 歴史的起源
多くは事故の反省から生まれた。特にTAB-02、TAB-09、TAB-10は人工悲劇後の再発防止と証拠保全に結びつく。

## 異説/矛盾
国家は安全規則として掲げるが、異端派は「禁忌は罪の跡を見せないための布」と批判する。
